# NoteHub (Next.js)

## Запуск проєкту

Встановлення залежностей:

npm install

Запуск у режимі розробки:

npm run dev

Відкрити в браузері:

http://localhost:3000

---

## Структура проєкту

app/ сторінки та маршрути (Next.js App Router)  
components/ перевикористовувані компоненти  
lib/ функції для роботи з API  
types/ типи та інтерфейси

---

## Маршрути

/ головна сторінка  
/notes список нотаток  
/notes/[id] сторінка деталей нотатки

---

## Опис

NoteHub — застосунок для створення, перегляду та пошуку нотаток.

Реалізовано:

- отримання списку нотаток
- пошук за ключовим словом
- перегляд деталей нотатки
- створення нової нотатки

---
